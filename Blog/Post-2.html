Ftp Guide

My go to FTP client is FileZilla, as it is Open source and avalible on MacOS, Linux, and Windows; and its really easy to set up as well!

I mainly use FTP to transfer ripped roms from my New 2DS xl to my pc as it makes it really convenient to not have to take out the Micro SD card every time. For that application I use FTPD

Step 1

Run your software of choice

Aquire server information, you will need the following:

  • Host - This will be a website or an IP address.
  • Port - this will be the number after an IP or Directory, Host.website.com:Port
  • Username - Username that you login to the server with. These are not always required. only really used to manage who gets acces to the server in the first place.
  • Password - On a server that needs a Username you will probably need a password with it.

    • Step 2

    Heres easy the part, Take all the provided information above into the corresponding boxes. Then press connect.

    FTP Connected!

    Step 3

    Download away! Its really that simple, if you truely wanted you could delve a lot deeper in regards to security and privacy but i just wanted to make a simple guide.

    I personally reccomend when doing any sort of P2P Downloading running a VPN however its nothing thats required.
